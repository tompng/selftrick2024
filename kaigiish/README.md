# Most Kaigi-ish

## Usage

```
IRBRC=./irb_theme_rk2024 irb
```

Or write `require '/path/to/irb_theme_rk2024.rb'` to your `~/.irbrc`.
You can also install with `gem install irb-theme-rk2024` and write `require 'irb/theme/rk2024'` to your `~/.irbrc`.

## Requirements

Reline >= 0.5
